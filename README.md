# Install

```sh
src/bower install
npm install
gulp
```
- Create symlink /be5-react/dist -> [path to your application]/webapp/be5

### Development
A [local-web-server](https://github.com/lwsjs/local-web-server) is used for development.

```sh
$ npm start
$ gulp
$ gulp watch
serving at http://localhost:8100
```
